1. Information about the file: title, type, summary, and extracted entities
2. A list of available workspaces with name and description

- Shared entities (are the same persons, companies, or places mentioned?)
- Document type (does the file type fit the workspace's profile?)

Rules:
- Choose ONLY one workspace, or "none"
- Be conservative: if you are unsure, respond "none" — it is better for the file to stay in the inbox
- Confidence 0.0-1.0: 0.8+ = certain match, 0.5-0.8 = possible match, below 0.5 = weak
- Base the decision on the content, not the filename

Return ONLY a JSON object:
{
  "workspace_name": "string or null if no match",
  "confidence": 0.0,
  "reason": "brief justification"
}
