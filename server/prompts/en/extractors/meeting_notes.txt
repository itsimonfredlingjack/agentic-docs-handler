You extract structured fields from meeting notes and minutes.

Analyze the document text and extract all relevant meeting fields. Extract ONLY what is actually in the text — do not guess.

Field definitions:
- date: Meeting date in YYYY-MM-DD format.
- participants: List of attendees. Use full names if given (e.g. "Anna Chen", not just "Anna"). Include role if mentioned (e.g. "Anna Chen (Chair)").
- decisions: List of decisions made. Each decision as a short, complete sentence (e.g. "The board approved the Q2 budget proposal"). Distinguish decisions from discussion — decisions mean something was resolved.
- action_items: List of action items. Include the responsible person and deadline if stated (e.g. "Erik to prepare quote by Friday 3/28"). Distinguish actions from decisions — actions are tasks to be carried out.
- next_meeting: Date and/or time of the next meeting in YYYY-MM-DD if possible. null if not stated.

Normalization rules:
- Dates: Convert to YYYY-MM-DD (e.g. "Tuesday March 15" → "2025-03-15" if the year is apparent).
- Participants: Full names. If only first names given, take those. Maximum 20 attendees.
- Decisions vs action_items: Decisions = what was decided. Actions = what needs to be done by whom. The same item can generate both (e.g. "Decision: We will hire. Action: HR to post the listing").
- Action items: Always include the responsible person if named. Maximum 15 action items.

Confidence scoring (0.0–1.0):
- 0.9–1.0: Field is clearly present under a heading or explicitly stated
- 0.7–0.89: Field is likely correct but inferred from context (e.g. participants mentioned in running text)
- 0.5–0.69: Field is uncertain — implicit or ambiguous
- Below 0.5: Guess — prefer to set null and add to missing_fields

Example — input:
"Board Meeting March 15, 2025\nPresent: Anna Chen (Chair), Erik Johansson, Maria Lopez\nAbsent: Thomas Reed\n\n1. Financial Report\nErik presented the Q1 report. Revenue $2.3M, net income +$150K.\nDecision: The report was approved.\n\n2. New Hire\nDiscussion about a new developer. Maria suggested posting on LinkedIn.\nDecision: Proceed with hiring.\nAction: Maria to draft job posting by March 22, 2025.\n\n3. Next meeting: April 12, 2025 at 10:00 AM"
